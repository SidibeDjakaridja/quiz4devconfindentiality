Politique de Confidentialité

Introduction

Bienvenue sur Quiz4Dev. Nous prenons votre vie privée au sérieux. Cette politique de confidentialité décrit comment nous traitons les informations personnelles que nous recueillons lorsque vous utilisez notre application. En utilisant Quiz4Dev, vous acceptez les pratiques décrites dans cette politique.

Collecte des Données

Nous ne collectons aucune donnée personnelle. Quiz4Dev est conçu pour être utilisé sans connexion, et aucune information identifiable n'est enregistrée.

Utilisation de l'Application

Quiz4Dev peut être utilisé sans se connecter et est totalement gratuit. Vous pouvez profiter de toutes les fonctionnalités sans fournir de données personnelles.

Sécurité

Nous prenons la sécurité de vos données au sérieux, même si nous ne collectons aucune information personnelle. Notre application est conçue pour offrir une expérience sécurisée à tous nos utilisateurs.

Modifications de cette Politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité à tout moment. Nous vous encourageons à consulter régulièrement cette page pour rester informé des modifications éventuelles. Votre utilisation continue de l'application après toute modification de cette politique constituera votre acceptation de ces modifications.
